# Python Mini-Projects

A collection of clean, interactive command-line applications built to practice core Python fundamentals, data handling, and control flow.

## 📝 Short Description
This repository contains three independent, terminal-based utility apps: a restaurant menu system, a student grade manager, and a task tracker. Together, they demonstrate how to capture user input, manipulate data structures, manage basic math calculations, and handle errors.

---

## 🛠️ Key Technical Highlights Included
* **Data Structures:** Uses Python lists for sequential tracking and dictionaries for key-value mapping[cite: 1, 2, 3].
* **Control Flow:** Implements conditional loops (`while True`) and conditional logic (`if-elif-else`) to build dynamic console menus[cite: 1, 2, 3].
* **Robust Input Handling:** Includes `try-except` blocks to catch data type errors and prevent application crashes[cite: 3].

---

## 📁 Project Portfolio Breakdown

### 1. Restaurant Bill Calculator (`hotelmenu.py`)
* **Purpose:** Allows a user to place an order from a predefined dictionary of items and instantly calculates the total bill[cite: 1].
* **Key Features:** Dynamic lookup matching, multiple order checking, and real-time total updates[cite: 1].

### 2. Student Grade Management System (`studentgrademanagment.py`)
* **Purpose:** A complete menu-driven console app to add, update, read, and delete student academic records[cite: 2].
* **Key Features:** Automated grading boundaries (A, B, C, Fail) and complete records management[cite: 2].

### 3. Task Management App (`todo.py`)
* **Purpose:** A classic to-do helper application that helps organize, add, modify, and delete active tasks[cite: 3].
* **Key Features:** Index tracking for task updates, clean output lists, and error validation[cite: 3].

## 🚀 How To Run These Projects

You can run these projects locally by copying and pasting the following commands into your terminal:

```bash

# 1. Clone the repository to your local machine
git clone [https://github.com/YOUR-GITHUB-USERNAME/Python-Mini-Projects.git](https://github.com/YOUR-GITHUB-USERNAME/Python-Mini-Projects.git)

# 2. Move into the repository folder
cd Python-Mini-Projects

# 3. Navigate into the projects directory
cd projects

# 4. Run any script of your choice
python todo.py

```
